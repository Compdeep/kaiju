04 / 09

依存性注入。

Dependency Injection.

JavaScript の 後で解決されるプロミスのような仕組みで、エージェントの事前計画を可能にする。プランナーが1回のLLM呼び出しでグラフ全体を生成し、上流ステップの完了とともに値が流れ込む。
JavaScript-like promises that resolve later enable upfront agent planning. The planner emits the whole graph in one LLM call — values flow in as their upstream steps finish.

05 / 09

Kaijuのあらゆる要素は、ノードである。

Every part of Kaiju is a node.

1つのスケジューラが全てのステップを実行する。あらゆる動作はノードとして表現され、リアルタイムで観測できる。
One scheduler runs every step — every action is a node, observable live.

IGX — セキュリティゲート。

IGX — the security gate.

呼び出し側がまずインテントを設定し、ツール実行時にゲートが強制する。
The caller sets intent first; the gate enforces it at tool-execution time.

スコープ Scope
ユーザーが呼び出せるツールの集合。
The set of tools a user is allowed to call.
インテント Intent
呼び出し側が決める破壊度の上限 — observe / operate / override。
A risk ceiling set by the caller — the model can't change it.
インパクト Impact
ツール作者が宣言する。bash('ls') と bash('rm -rf /') は別物。
Declared by the tool author — param-aware.
クリアランス Clearance
あなたのHTTPエンドポイントが許可/拒否を返す。
An HTTP endpoint you control — your policy, not ours.
